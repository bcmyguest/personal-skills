# frontend

| Skill | What it does |
|-------|--------------|
| `react-ts-setup` | Scaffold a new React + TypeScript SPA with Vite — asks the key decisions up front (package manager, Node manager + version, UI framework + theme colours, CI provider, dev Docker), then wires React Compiler, ESLint + Prettier, Vitest, husky + lint-staged + pre-commit, editor settings, env templates, Docker, CI, and a README, keeping the Node version in lockstep and leaving lint/typecheck/test/build green. |
| `senior-frontend-refactor` | Refactor frontend code as a senior engineer — improve structure, readability, performance, and test coverage while preserving behavior. |

Skills live in [`../skills/`](../skills); this plugin selects them via `skills` in the
[marketplace manifest](../.claude-plugin/marketplace.json).
